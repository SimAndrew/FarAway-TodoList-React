### Run the app:

- Clone a project:

```
https://github.com/SimAndrew/FarAway-TodoList-React.git
```

- Open project code in your editor.
- Install the dependencies, enter into the terminal:

```
npm install
```

- Run the project, enter into the terminal:

```
npm run dev
```
